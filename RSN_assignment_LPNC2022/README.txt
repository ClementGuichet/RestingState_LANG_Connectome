Assignment of Power 264 ROIs to CAB-NP Atlas networks (Ji et al., 2019)
----------------------------------------------------------------------------------------------------------
- Atlas alignment
- Multiplication to get the network labels for each voxel for each ROI
- Network overlap calculator to get the exact composition of each ROI in terms of RSN

An MNI-space projected volumetric version of CAB-NP parcellation can be found at this link (Smine & Orban, 2022):
https://figshare.com/articles/dataset/CAB-NP_projected_on_MNI2009a_GM_volumetric_in_NIfTI_format/14200109
